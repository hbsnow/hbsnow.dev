# Blockcode

コードを記述します。

<Preview>
  <Story name="default">
    <Container>

`language` に言語を使用することでシンタックスハイライトを有効にします。この属性を明示的に付与しない限り、シンタックスハイライトは有効になりません。

<Preview>
  <Story name="language">
    <Container>
      <Margin all={2}>
        <Blockcode language="js">
          const message = "Blockcode with language"
        </Blockcode>
      </Margin>
    </Container>
  </Story>
</Preview>
