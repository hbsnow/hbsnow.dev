import { Meta, Story, Preview } from '@storybook/addon-docs/blocks'

import Container from '../container/Container'
import Margin from '../margin/Margin'
import Rating from './Rating'

<Meta title="Rating" component={Rating} />

# Rating

初期値で上限が 3 までのレーティーング用のスターです。`rate` の数値分スターがつきます。

<Preview>
  <Story name="rate">
    <Container>
      <Margin all={2}>
        <Rating rate={0} />
      </Margin>
      <Margin all={2}>
        <Rating rate={1} />
      </Margin>
    </Container>
  </Story>
</Preview>

`maxRate` を指定すると上限値が変わります。

<Preview>
  <Story name="maxRate">
    <Container>
      <Margin all={2}>
        <Rating rate={1} maxRate={2} />
      </Margin>
      <Margin all={2}>
        <Rating rate={5} maxRate={10} />
      </Margin>
    </Container>
  </Story>
</Preview>
